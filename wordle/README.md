# 🟩 Wordle - 英単語推測ゲーム

本家Wordleを忠実に再現した、ブラウザで遊べる5文字英単語推測ゲームです。

## 🎮 遊び方

1. 5文字の英単語を入力して推測します
2. 各文字のタイルの色が変わり、ヒントが表示されます：
   - 🟩 **緑** — 正しい文字が正しい位置にある
   - 🟨 **黄** — 正しい文字だが位置が違う
   - ⬛ **灰** — 単語に含まれない文字
3. 6回以内に正解を当てましょう！

## ✅ 実装済み機能

- **ゲームボード** — 6×5のタイルグリッド
- **オンスクリーンキーボード** — タッチ操作対応
- **物理キーボード対応** — PC/Macのキーボード入力
- **3041語のターゲット単語** — ランダム出題（2300語以上の要件を大幅達成）
- **4009語のバリデーション辞書** — 推測語の有効性チェック
- **タイルアニメーション** — フリップ、バウンス、シェイク
- **キーボード色反映** — 推測結果に応じてキー色が変わる
- **正解発表モーダル** — ゲーム終了時に結果表示
- **🔍 Google検索リンク** — 正解単語の意味をワンクリックで検索
- **統計機能** — プレイ数、勝率、連勝数、推測回数分布
- **結果共有** — 絵文字でクリップボードにコピー
- **ダークモード** — 背景色切り替え
- **ハイコントラストモード** — 色覚多様性に対応
- **ハードモード** — ヒントで得た文字を必ず使用
- **ヘルプモーダル** — 遊び方の説明
- **レスポンシブデザイン** — スマホ・タブレット対応
- **ゲーム状態保存** — LocalStorageで進行状態を記憶
- **New Game** — ランダムな新しい単語で再挑戦

## 🌐 アクセスURL

## 💾 データ管理

- **LocalStorage** を使用
  - `wordle_daily` — 現在のゲーム状態（単語、推測、評価）
  - `wordle_stats` — 統計データ（プレイ数、勝率、連勝等）
  - `wordle_settings` — 設定（ダーク/ハイコントラスト/ハードモード）
  - `wordle_visited` — 初回訪問フラグ
