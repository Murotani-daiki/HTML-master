ループ処理

コンピュータの得意分野：面倒な繰り返し作業を自動化しよう

ループ処理とは？：面倒な繰り返しを自動化する

プログラミングを学ぶ上で、最も大きな恩恵を感じる瞬間の一つが「ループ処理（繰り返し）」をマスターした時でしょう。「1万個のデータに名前をつける」「同じ計算を100万回繰り返す」といった、人間が行えば一生かかるような作業を、コンピュータはたった数行のコードで、しかも一瞬で完了させます。

Pythonのループ処理は非常に直感的で読みやすく設計されています。データ処理、Webスクレイピング、AIの学習プロセスなど、現代のあらゆるテクノロジーの基盤となる技術です。

1. for 文：イテラブル（反復可能）な要素を回す

Pythonの `for` 文は、リストや文字列などの「要素の集まり」から、中身を一つずつ順番に取り出して処理を行うのが得意です。

Python：リストのループ 
fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits:
    print(f"フルーツの名前: {fruit}")

ここで登場する `fruit` は、リストの中身が代入される一時的な変数です。この変数を自由に使うことで、各要素に対して一括で処理をかけることができます。

2. range() 関数：回数を指定して繰り返す

「10回繰り返したい」「1から100まで足し合わせたい」といった、数値をベースにした繰り返しには `range()` 関数を使います。

range() の柔軟な引数パス

  • range(stop): 0から(stop-1)まで繰り返す。
  • range(start, stop): startから(stop-1)まで。
  • range(start, stop, step): stepの間隔で数値を刻む。
Python：rangeの活用 
# 0, 2, 4, 6, 8 と表示される
for i in range(0, 10, 2):
    print(i)

3. ループの制御：break と continue

ループの途中で逃げ出したり、特定の回だけ処理を飛ばしたりすることも可能です。

  • break: ループを完全に終了し、外に出ます。
  • continue: 今行っている回だけを中断し、次の回へ進みます。
Python：ループの脱出 
for i in range(1, 10):
    if i == 5:
        break # 5になったら強制終了
    print(i)

4. Python特有の機能：ループ後の else

Pythonのユニークな機能として、ループの後に `else` を書くことができます。これは、**「一度も break せずにループを最後まで完走した時のみ」** 実行されます。

Python：for-else文 
for user in user_list:
    if user == "Target":
        print("見つけました！")
        break
else:
    print("最後まで見つかりませんでした。")

5. while 文：条件が満たされるまで繰り返す

繰り返す回数が決まっていない時、例えば「正しいパスワードが入力されるまで」「ユーザーが終了ボタンを押すまで」といった場合には `while` が適しています。

Python：whileループ 
energy = 3
while energy > 0:
    print(f"行動中... 残りパワー: {energy}")
    energy -= 1
print("エネルギー切れです")

🚀 パフォーマンスの知恵：range関数は「軽い」

Python 3の `range()` は、どれほど大きな範囲を指定しても大量のメモリを消費しません。なぜなら、100万個のリストを作るのではなく、ループのたびに「次の数値」をその場で作っているからです。非常に効率的ですね！

よくある質問 (Q&A)

Q: 無限ループになってしまいました！

A: 落ち着いて `Ctrl + C` を押してください。プログラムを強制終了できます。`while` 文を書く時は、必ず「いつかは終わる条件」になっているか確認しましょう。

Q: インデックス番号も一緒に取得したい時は？

A: `enumerate()` 関数を使うのがスマートです。`for i, fruit in enumerate(fruits):` と書くことで、番号 `i` と中身 `fruit` を同時に取り出せます。

Q: 逆順に回したい時は？

A: `reversed()` 関数を使うのが簡単です。 `for item in reversed(items):` とするだけで、リストの後ろから順に取り出すことができます。
