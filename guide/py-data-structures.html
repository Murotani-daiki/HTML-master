データ構造

情報をひとまとめにして管理し、効率よくプログラムを動かそう

情報をどうまとめるか？：データ構造の重要性

プログラムでは単一の数値や文字列だけでなく、「複数のデータの集まり」を扱うことがほとんどです。Pythonには、用途に合わせて使い分けられる強力なデータ構造（コンテナとも呼ばれます）が標準で備わっています。

適切なデータ構造を選ぶことは、プログラムの「読みやすさ」だけでなく「実行速度」にも直結します。名簿を作るなら辞書、履歴を保存するならリスト、といった具合に、それぞれの特性を理解することがプロへの第一歩です。

リスト [list]

順番（インデックス）のあるデータの集まり。中身の書き換えが自由自在です。

辞書 {dict}

「名前（キー）」と「値」をセットで管理。検索スピードが驚異的に速いです。

1. リスト [list]：並び順が命の万能選手

最も頻繁に使われるデータ構造です。カッコ `[ ]` で囲み、カンマで区切ります。

Python：リストの操作 
colors = ["Red", "Green", "Blue"]

colors.append("Yellow") # 末尾に追加
colors[0] = "Crimson"   # 0番目を書き換え
print(colors[1:3])      # スライス：1番目から2番目まで取得

リストの強力な機能の一つが **スライス** です。 `[開始:終了]` と書くことで、リストの一部を一気に切り出すことができます。

2. 辞書 {dict}：膨大なデータから一瞬で見つけ出す

キー(Key)と値(Value)を組み合わせて保存します。数万件のデータがあっても、キーを指定すれば一瞬で値を取り出せるのが最大の特徴です（ハッシュマップ構造）。

Python：辞書の活用 
user_scores = {"Alice": 95, "Bob": 80, "Charlie": 70}

# データの取得
print(user_scores["Alice"])

# データの追加・更新
user_scores["David"] = 85

3. タプル (tuple) と 集合 {set}

これら二つも、特定の場面で非常に強力な力を発揮します。

タプル：不変(Immutable)なリスト

丸カッコ `( )` を使います。最大の特徴は、一度作ると **「中身を変更できない」** ことです。
例: `coordinate = (35.6895, 139.6917)` # 座標など、変わっては困るデータに。

集合：重複を許さない、数学的な集まり

波カッコ `{ }` を使います。重複する値は自動的に一つにまとめられます。
例: `tags = {"Python", "HTML", "Python"}` # 重複が消え `{"Python", "HTML"}` になる。

4. Pythonの魔法：リスト内包表記

Pythonには、既存のリストから新しいリストを一行で作る非常にエレガントな書き方があります。

Python：リスト内包表記 
numbers = [1, 2, 3, 4, 5]
# すべてを2倍にした新しいリストを作る
doubled = [n * 2 for n in numbers]
# print(doubled) -> [2, 4, 6, 8, 10]

🚀 パフォーマンスの豆知識：計算量

・リストから特定の値を「探す」のは、データ量が増えるほど時間がかかります。
・辞書からキーを使って値を「探す」のは、データがどれだけ増えても常に一定の速さです。
賢いデータ構造の選択が、爆速のアプリを生み出します。

よくある質問 (Q&A)

Q: リストの中に別のリストを入れることはできますか？

A: 可能です。これを「多次元リスト（ネスト）」と呼びます。表形式のデータや、ゲームのマップ情報（X, Y座標）などを表す際によく使われます。

Q: 辞書のキーにリストを使うことはできますか？

A: できません。辞書のキーには「変更不可能なデータ（不変）」である必要があります。そのため、リストの代わりにタプルをキーとして使うことは可能です。

Q: リストを並び替えるには？

A: `list.sort()` メソッドを使うと中身を直接書き換えてソートします。元のリストを壊したくない場合は `sorted(list)` 関数を使いましょう。

まとめ

それぞれのデータ構造には得意・不得意があります。データの順序が大事なら **リスト**、名前で管理したいなら **辞書**、不変性を保ちたいなら **タプル**、重複を除きたいなら **集合**。これらを意識して使い分けることで、あなたのPythonコードはより洗練されたものになります！
