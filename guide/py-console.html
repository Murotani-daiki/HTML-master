出力とデバッグ

print()：プログラムの実行結果を画面に表示しよう

print() 関数：プログラムの「声」を聞く

プログラムの中で起きていることは、通常、コンピュータのメモリという目に見えない場所で処理されています。その中身を私たちの目に見える形にする魔法の道具が `print()` 関数です。

「正しく計算できているか？」「どこまで処理が進んだか？」といった確認作業（デバッグ）において、`print()` はプログラマーにとって最も身近で強力な相棒となります。Pythonの学習において、最初から最後までお世話になる、最も重要な関数の一つです。

1. 基本から一歩進んだ使い方

ただ一つの値を出すだけでなく、カンマ `,` で区切ることで複数の情報を一度に表示できます。その際、Pythonは自動的に単語の間に半角スペースを挿入してくれます。

Python：基本的な出力 
name = "Python"
score = 100
print("言語:", name, " | スコア:", score)
# 結果: 言語: Python | スコア: 100

出力をカスタマイズする引数

`print()` には、出力を細かく制御するための隠れたオプションがあります。

  • sep (separator): 区切り文字を指定します。デフォルトは半角スペースです。
  • end: 出力後の末尾を指定します。デフォルトは改行 (`\n`) です。
Python：sepとendの活用 
# 区切りをハイフンにする
print("apple", "orange", "banana", sep="-") 
# apple-orange-banana

# 改行せずに繋げる
print("Hello", end="")
print("World")
# HelloWorld

2. 文字列フォーマットの進化：f-strings

現代のPython（3.6以降）で最も推奨されるのが、**f文字列 (f-strings)** です。文字列の先頭に `f` をつけ、入れたい変数を `{ }` で囲むだけで、驚くほど直感的に文章を組み立てられます。

Python：f-stringsの威力 
user = "管理者"
count = 5
print(f"ようこそ、{user}さん。未読メッセージが{count}件あります。")

数値を綺麗に整える

f-stringsを使うと、桁数の指定や小数点の丸め込みも一瞬で行えます。

Python：書式指定 
pi = 3.14159265
print(f"円周率は約{pi:.2f}です") # 小数点第2位まで (3.14)

price = 1280000
print(f"価格: {price:,}円")       # カンマ区切り (1,280,000)

3. デバッグの裏技：f"{var=}"

Python 3.8から導入された最強のデバッグ機能がこれです。変数名の後に `=` をつけるだけで、「変数名とその中身」を同時に出力してくれます。

Python：爆速デバッグ 
x = 10
y = 20
print(f"{x=}, {y=}")
# 結果: x=10, y=20 (いちいち文字を書く手間が省ける！)

💡 昔の書き方に遭遇したら

古い教材やコードでは `%` を使う方法や `.format()` を使う方法が見られます。現在は `f-strings` が最も高速で推奨されていますが、これらを見かけても「古い書き方だな」と理解できれば十分です。

よくある質問 (Q&A)

Q: 画面ではなく「ファイル」に書き出すことはできますか？

A: 可能です！ `print("text", file=open("log.txt", "w"))` のように `file` 引数を使うことで、コンソールではなくファイルへ直接書き出すことができます。

Q: print() で改行したくない時は？

A: `end=""` を指定してください。これで次の `print()` の内容が同じ行に続けて表示されるようになります。

Q: 複雑なリストや辞書を見やすく表示するには？

A: Python標準の `pprint` モジュールを使うのがお勧めです（Pretty Printの略）。階層構造を綺麗にインデントして表示してくれます。

まとめ

「バグかな？」と思ったら、まずは迷わず `print()` を使って中のデータを確認しましょう。f-stringsを使いこなし、データの変化をリアルタイムで「見る」習慣をつけることが、脱・初心者への最短ルートです。あなたのプログラムが何を考えているのか、`print()` を通じて対話を楽しんでください！
